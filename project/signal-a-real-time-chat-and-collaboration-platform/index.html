Kendozone: Create tournaments in instants

Créez des tournois, invitez des compétiteurs, et générez les arbres de compétition selon l’algorithme choisi.

 

  • Création et configuration de tournois
  • Créer et configurer des championnats en fonction de la catégorie
  • Invitation en masse ou ajout manuelle de concurrents
  • Génération d’arbres (basé sur le plugin Laravel tournament )
  • Gestion des équipes
  • Génération de documentation : Liste des combats, feuilles de score
  • Gestion des compétiteurs / Clubs / Associations / Fédérations
  • Multilingue : Traduit en 4 langues : Anglais, Français, Espagnol, Japonais. Aide à la traduction

Stack : PHP7 / Laravel 5.7 / MySQL  / JQuery

Vous pouvez consulter ce projet Open Source sur Github : Kendozone

Il existe un projet inachevé de migration à Angular 6 : kz-front, kz-api
